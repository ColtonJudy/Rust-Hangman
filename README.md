This program is a simple command line hangman game written in Rust.

## To Install

1. Install Rust: https://www.rust-lang.org/tools/install

2. Clone Repository
   ```bash
   git clone https://github.com/ColtonJudy/Rust-Hangman
   cd .\Rust-Hangman\

3. Cargo Build
   ```bash
   cargo build
   
5. Cargo Run
   ```bash
   cargo run
